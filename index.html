IOT(Internet Of Thing)

  • The internet of things, or IoT, is a network of interrelated devices that connect and exchange data with other IoT devices and the cloud.

  • The field has evolved due to the convergence of multiple technologies

  • Access to low-cost, low-power sensor technology.

  • Machine learning and analytics.

  • Conversational artificial intelligence (AI).

  • Electric Devices are directly connected and communicate with a controller computer, such as a cell phone, resulting in efficient electricity use. As a result, there will be no unnecessary use of electricity equipment.

Web-Development

  • Web development, also known as website development, refers to the tasks associated with creating, building, and maintaining websites and web applications that run online on a browser.

  • The basic tools involved in web development are programming languages called HTML (Hypertext Markup Language), CSS (Cascading Style Sheets), and JavaScript.

  • Frontend & React Frontend & React

  • Mobile Front-End Web Developer

  • Back-End Development

UI/UX Design

  • User experience (UX) refers to the user’s journey when interacting with a product or service.

  • UI design, on the other hand, refers to the actual interfaces with which users engage.

  • Designing research instruments

  • UI designers oversee the specifics of a product or service’s interface

  • Color scheme and font selection.

  • Containers help you group page elements and hold content together in cohesive units.

Flutter


  • Flutter is an open source framework by Google for building beautiful, natively compiled.

  • Flutter ships applications with its own rendering engine which directly outputs pixel data to the screen

  • The first version of Flutter was known as "Sky" and ran on the Android operating system.

  • Dart language, Flutter apps are written in the Dart language.

  • Flutter's engine, written primarily in C++, provides low-level rendering support using either Google's Skia graphic.

Android

  • Android development is the process of creating applications for devices running an Android operating system.
  • Android Studio is the official integrated development environment (IDE) for native Android.
  • APIs and cloud services: The Android platform offers a wide variety of APIs that can help you tap into the cloud and overcome the limitations of mobile devices.
